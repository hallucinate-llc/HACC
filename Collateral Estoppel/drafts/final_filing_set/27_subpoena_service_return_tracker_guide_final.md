## How to use

1. Start each recipient row when service is completed.
2. Enter production due date from subpoena face.
3. Mark whether checklist and search report were received.
4. If incomplete, mark deficiency notice date and cure deadline.
5. If cure fails, mark motion-to-compel filing date.

## Field notes

- checklist_received: Y/N
- search_report_received: Y/N
- privilege_log_received: Y/N/NA
- deficiency_notice_sent: date notice sent
- cure_deadline: date/time provided in deficiency notice
- cure_received: Y/N with date in notes
- motion_to_compel_filed: filing date

Use this tracker together with certificate of service and filed court records.
