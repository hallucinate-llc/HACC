# Proof-Gap Matrix: Housing And Probate Theories

This note isolates the current proof posture for the main theories being used in the housing and probate filings.

It is designed to answer four questions for each theory:

1. what duty or issue is being asserted;
2. what the current best support is;
3. what the weakest link is; and
4. what additional document or record would most directly close the gap.

## Use Notes

- Treat the left-most housing theories as the strongest current branches.
- Treat Quantum contract-duty expansion, race-comparator proof, and true collateral estoppel as the most proof-sensitive branches.
- When a motion needs to stay tight, prefer:
  - HACC law-first duties
  - HACC-directed option administration
  - packet-routing failure
  - writ / notice / tailoring objections in probate
- When a judge asks "what would close that gap," use the last column rather than arguing abstractly.
