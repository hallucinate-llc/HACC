## Intake questions

Do not promote the predicate unless all of these can be answered from filing-specific record support:

1. Can the challenged filing be linked to specific conduct suggesting delay, pressure, or collateral control?
2. Is that linkage supported by staged documents or communications rather than only by generalized suspicion?
3. Can the theory be stated without depending on still-unresolved issue-preclusion or prior-authority assumptions?
4. Is the filing-specific motive record strong enough that the branch can be described as mapped, not merely investigated?
5. Can the note identify the concrete supporting sources for the improper-purpose inference?

The safest minimum package is:

1. one updated improper-purpose worksheet entry in [50_orcp17_improper_purpose_worksheet_2026-04-07.md](/home/barberb/HACC/Collateral%20Estoppel/drafts/final_filing_set/50_orcp17_improper_purpose_worksheet_2026-04-07.md)

## Do not promote if

Leave `improper_purpose_mapped` as `false` if any of the following remains unresolved:

1. the theory still depends mainly on suspicion or rhetoric;
2. the source record still shows only generalized conflict, not filing-specific improper purpose;
3. the strongest support still depends on unresolved issue-preclusion or actor-attribution gaps;
4. the record is not yet strong enough to describe the branch as mapped.
