# Clackamas FTR Request Path Note - 2026-04-08

This note records the current official Clackamas County Circuit Court path for requesting hearing audio.

## Official Clackamas sources

1. Records request forms page:
   - https://www.courts.oregon.gov/courts/clackamas/help/pages/records-requests-forms.aspx
2. Audio transcript requests page:
   - https://www.courts.oregon.gov/courts/clackamas/records/pages/atr.aspx
3. Direct PDF form:
   - https://www.courts.oregon.gov/courts/clackamas/records/Documents/FTR%20Req%20Form%20_Rev%2005.2022.pdf

## What the Clackamas page says

The court states that to request a copy of an audio recording of a hearing or trial, the request must include:

1. date of the proceeding;
2. judge who heard the case;
3. courtroom number, if known;
4. case name;
5. case number;
6. requester name, address, and phone number;
7. email address for electronic delivery.

The current Clackamas page also states:

1. electronic recording: `$10` per request / hearing / day;
2. CD recording: `$11` per request / hearing / day;
3. payment must be made in full before processing;
4. records unit contact: `503-655-8447` with option `7` for questions.
