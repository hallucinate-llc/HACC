## Operational Rule

When the same underlying source file appears again in another binder family:

1. keep the full printed source in the `Shared Housing / State-Court Binder` when that is the master merits use;
2. keep the later exhibit's tab cover page and exhibit cover page;
3. replace the repeated exhibit body with a short cross-reference page stating that the source is incorporated by reference from the state housing/state-court binder; and
4. identify the distinct proposition the later exhibit is still being used to prove.

## Current Master-By-Reference Uses

The current lean/reference mode treats these as incorporated by reference to the master state housing/state-court binder:

1. `Shared Housing / State-Court Binder - Exhibit X`
   - same underlying `JC Household` chain as `Exhibit V`
2. `Shared Housing / State-Court Binder - Exhibit AA`
   - same underlying `JC Household` chain as `Exhibit V`
3. `Shared Motion / Probate / Sanctions Binder - Exhibit W`
   - same underlying `JC Household` chain as `Shared Housing / State-Court Binder - Exhibit V`
4. `Supplemental Background Appendix - Exhibit B-1`
   - same underlying Waterleaf application artifact as `Shared Housing / State-Court Binder - Exhibit W`

## Best Use

Use this approach when:

1. the binders will be carried together;
2. the state housing/state-court binder is already present as the master common volume; and
3. the goal is to reduce repeated exhibit bodies while preserving proposition-level tracking.
