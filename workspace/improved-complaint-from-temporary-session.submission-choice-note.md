# Submission Choice Note

Use this note to decide whether to file the reduced packet or the full packet first.

## Use The Reduced Packet If

- You need to file quickly.
- You want the cleanest first-wave set centered on the strongest current evidence.
- You want to avoid overloading the initial filing with exhibits that are helpful but not essential.
- You want the court to see the core notice, retaliation, accommodation, and race-contracting sequence first.

## Use The Full Packet If

- You want the court to have the full displacement, notice, lease, voucher, and accommodation chronology at the outset.
- You expect the defendants to argue missing context or incomplete sequence.
- You are comfortable submitting a larger exhibit package now.

## Best Current Recommendation

The strongest practical first move is the reduced packet.

Why:
- It contains the complaint plus the best current documentary anchors.
- It keeps Exhibit Q in the right limited role.
- It keeps Exhibit R front and center as direct proof of the February 26, March 2, and March 9, 2026 complaints.
- It now also includes Exhibit S as a concise preserved record of the SHS race-conscious outcome-goal language cited in the complaint.
- It avoids burying the strongest counts inside a larger exhibit stack.

## If You Add A Declaration

- Only add a declaration if it is signed and fully truthful.
- The Benjamin Julio declaration is most useful if Count V or the hearing-access issue is a current filing priority.
- If no declaration is ready, the packet is still stronger filed cleanly without one than filed with an unfinished or overstated declaration.

## Final Safe Framing

- Reduced packet first is the best option for speed and clarity.
- Full packet remains available if you want the entire record attached from the start.
