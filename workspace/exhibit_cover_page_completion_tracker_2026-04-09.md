# Exhibit Cover Page Completion Tracker

Use this tracker to make sure **every exhibit** receives:

1. a `tab cover page`; and
2. an `exhibit cover page`

before it is placed into any binder, appendix, or filing packet.

## Governing Rule

No exhibit should be treated as binder-ready unless all three layers exist:

1. tab cover page
2. exhibit cover page
3. exhibit itself
