# Exhibit Cover Page Assembly Note

Use this note when building any filing binder or exhibit booklet.

## Required Order For Each Exhibit

1. `Tab cover page`
2. `Exhibit cover page`
3. `Exhibit itself`

## The Tab Cover Page Must Show

- exhibit label
- section or binder family
- short title
- source file
- status (`original`, `true copy`, `mailbox copy`, `OCR / text scan`, `placeholder pending production`)

## The Exhibit Cover Page Must Show

- exhibit label
- short title
- source file
- each motion, declaration, or memorandum that relies on it
- the proposition(s) the exhibit is supposed to prove
- authenticity / foundation note
- limitation note if the exhibit is partial, mailbox-derived, OCR-only, or demonstrative
