# Bench Card: Eviction Response And Interim Relief

## Core Theme

HACC should not get possession while both housing paths it directed the household to use were still unfinished:

- the Hillside Park redevelopment replacement-housing application path was not processed;
- the alternate port-out path to Multnomah County stretched across four calendar months and still was not complete.

## Best 30-Second Opening

Your Honor, this is not a simple holdover. HACC triggered redevelopment-related relocation duties, told us to use a replacement-housing application path tied to Hillside Park, and also told us in the alternative to port the voucher to another county. The first path was not processed, and the second path remained incomplete across four calendar months. HACC should not get possession while the very paths it directed us to use were still unfinished.

## Five-Step Story

1. HACC triggered relocation duties.
2. HACC directed the household into the Hillside Park replacement-housing application route.
3. That packet was not processed and not forwarded.
4. HACC also identified porting to Multnomah County, but that route remained incomplete across four calendar months.
5. HACC nevertheless moved toward eviction.

## Best Legal Point

This case should not be treated as a household-caused failure to move where the record shows the housing paths HACC itself identified remained unfinished.

## Narrow Ask

- deny possession as premature;
- or at minimum stay / abate possession;
- require a written status update within 7 days;
- preserve the application, voucher, transfer, and relocation path.

## If The Judge Pushes Back

`What exactly was not processed?`

- The Hillside Park replacement-housing application path through the Blossom / Hillside Manor intake route.

`What about the other county option?`

- The Waterleaf / Multnomah County port-out route remained incomplete from December through late March.

`Are you asking for a specific apartment?`

- No. Only preservation of the housing path and no possession while relocation remains incomplete.
