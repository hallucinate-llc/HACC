# Exhibit Cover Page Master Tracker

Use this tracker as the source document for preparing:

1. a tab cover page; and
2. an exhibit cover page

for each current exhibit family.

The governing rule is:

- every exhibit gets a `tab cover page`;
- every exhibit gets an `exhibit cover page`;
- the exhibit cover page must identify:
  - which motion, declaration, or memorandum relies on it; and
  - the proposition the exhibit is supposed to prove.

## 5. Assembly Rule

For every exhibit packet:

1. place a tab cover page first;
2. place an exhibit cover page second;
3. place the exhibit itself third;
4. if the exhibit is a placeholder, breadcrumb, OCR, mailbox copy, or demonstrative, say so on the exhibit cover page.
