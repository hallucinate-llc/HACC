# Court-Ready Exhibit Binder

This folder is the working area for the state-court exhibit binder.

## Folder Layout

- `covers/`
  One cover sheet per exhibit.
- `exhibits/`
  The actual exhibit files to be lodged or attached.
- `compiled/`
  Final merged or print-ready binder outputs.
- `working/`
  Temporary notes, OCR extracts, and page-targeting work product.

## Binder Rule

Each exhibit should have a matching cover sheet that states:

1. the exhibit number and title;
2. the source file path;
3. the page or pages within the source document that matter;
4. the factual assertion or legal point the exhibit supports; and
5. the page, section, or paragraph in the filing where that assertion appears.

## Naming Convention

- Cover sheet:
  `covers/Exhibit_A_cover_sheet.md`
- Exhibit file copy or pointer note:
  `exhibits/Exhibit_A_<short_title>.pdf`

## Minimum Cover-Sheet Fields

- `Exhibit Number`
- `Exhibit Title`
- `Source File`
- `Relevant Source Page(s)`
- `Assertion Supported`
- `Used In Filing`
- `Filing Page / Section / Paragraph`
- `Why This Exhibit Matters`
