## SECTION 3: STEP-BY-STEP CUSTOMIZATION PROCESS

### Step 1: Open the Motion File in Your Text Editor

1. Open file: `/home/barberb/HACC/Breach of Contract/outputs/joinder_quantum_001_motion_for_joinder.md`
2. In your word processor, select **Edit → Find & Replace** (Ctrl+H)
3. A dialog box will open with "Find" and "Replace" fields

### Step 2: Replace Placeholders One by One

**Starting with [CASE NUMBER]:**
1. In "Find" field, type: `[CASE NUMBER]`
2. In "Replace" field, type: (e.g.) `2024CV15234`
3. Click **Replace All** (replaces all instances at once)
4. **Check:** Close dialog, scroll document to verify replacement looks correct

**Repeat for each placeholder:**
- Click Find & Replace again
- Enter next placeholder in "Find" field
- Enter your value in "Replace" field
- Click Replace All

### Step 3: Verify All Replacements

After each replacement, scroll through document and check visually:
- [ ] No [BRACKETS] remain visible
- [ ] Replaced text makes sense in context
- [ ] Case number appears correctly on first page
- [ ] Judge name appears correctly
- [ ] Attorney names spelled correctly
- [ ] Addresses are complete

### Step 4: Save Customized Version

1. **File → Save As**
2. New filename: `joinder_quantum_motion_[YOUR_CASE_NUMBER].doc` (or .docx or .pdf)
   - Example: `joinder_quantum_motion_2024CV15234.doc`
3. Save to clear location on your computer (e.g., Desktop or "HACC Motion" folder)

## SECTION 4: FORMATTING TIPS (For Professional Appearance)

Once customized, before converting to PDF, ensure formatting meets Oregon court standards:

### Font & Spacing
- [ ] Font: Times New Roman, 12 point
- [ ] Line spacing: Double-spaced (1.5 minimum for headers/sections)
- [ ] Margins: 1 inch on all sides
- [ ] Page numbers: Bottom center of each page (except first page number optional)

### Header & Title
- [ ] Case name centered at top of each page
- [ ] Case number on right or centered
- [ ] "MOTION FOR JOINDER OF QUANTUM RESIDENTIAL PROPERTY MANAGEMENT" centered, bold

### Exhibits
- [ ] Each exhibit labeled "EXHIBIT A", "EXHIBIT K", "EXHIBIT L", "EXHIBIT M"
- [ ] Label appears at top of first page of each exhibit
- [ ] Page numbers continue sequentially (e.g., motion ends page 6, Exhibit A starts page 7)

### Final Check Before Filing
- [ ] No typos or grammatical errors
- [ ] All case-specific names (judge, counsel, parties) spelled correctly
- [ ] All legal citations are correct and formatted consistently
- [ ] Signature block is signed (in blue ink if printing originals)

## SECTION 5: COMMON MISTAKES TO AVOID

❌ **MISTAKE 1: Forgetting to Replace All Instances**
- Finding "Judge Smith" is on page 1 — but checking page 5 and finding "[JUDGE NAME]" still there
- **FIX:** Use "Replace All" rather than "Replace" one-by-one; then scroll entire document to verify

❌ **MISTAKE 2: Replacing Placeholders Inside Quotes or Examples**
- Example from motion reads: "[Under ORCP 29A, the standard test is...]" and you replace "[" with your case number by mistake
- **FIX:** Be precise with placeholder text; include the brackets: `[JUDGE NAME]` not just `JUDGE NAME`

❌ **MISTAKE 3: Changing Legal Citations or Authority Language**
- You customize placeholders for judge/attorney names, but accidentally delete or modify legal text (e.g., "ORCP 29A" becomes "ORCP 99A")
- **FIX:** Only replace [BRACKETED] placeholders; never modify text in quotes, legal citations, or argument sections

❌ **MISTAKE 4: Wrong Address or Phone Number**
- You paste attorney contact info from old case file; phone/email don't match HACC counsel
- **FIX:** Verify each piece of contact info by calling the person or checking court docket AFTER replacing

❌ **MISTAKE 5: Printing Old Version by Accident**
- You customize motion in Word, save it, but accidentally print the original PDF instead of new version
- **FIX:** Use consistent naming (e.g., "motion_CUSTOMIZED_[DATE].pdf"); delete old versions after confirming new one is correct

## SECTION 8: MOVING FROM WORD TO PDF (Final Step)

Once customized and verified:

### If Using Microsoft Word:
1. File → Print
2. Select printer: "Microsoft Print to PDF" (or "Print to PDF")
3. Click Print
4. Save dialog appears: Name the file `joinder_quantum_motion_[CASE_NUMBER].pdf`
5. Click Save
6. Open PDF in Adobe Reader to verify formatting looks professional

### If Using Google Docs:
1. File → Download → PDF Document
2. Save file to your computer with clear name
3. Open in Adobe Reader to verify

### If Using LibreOffice/OpenOffice:
1. File → Export as PDF
2. Verify PDF options (standard settings are fine)
3. Click Export
4. Save with clear name

## FINAL REMINDER

✅ **You don't need to understand all the legal arguments in the motion.** Your job is simply to replace the [BRACKETED] terms with your actual case information.

✅ **The legal text stays exactly the same.** You're not editing the law or the arguments — just plugging in your case specifics.

✅ **If you're unsure about a placeholder value, look it up or call the court.** Better to ask the clerk "What is the judge's full name?" than to file with incorrect info.

✅ **Test your Find & Replace on a COPY.** Save a backup of the original before customizing, so if you make a mistake, you can start over.

✅ **When in doubt, ask an attorney.** If you're self-representing and unsure whether to change something, it's safer to ask for 5 minutes of attorney time than to file something incorrect.
