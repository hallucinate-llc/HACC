# Clerk Filing Bundle Sequence (Quantum Joinder Packet)

Use this as the upload order for e-filing so the docket reads clearly and the judge sees the packet in logical sequence.

## Suggested Docket Event Order

1. Motion for Joinder and Leave to File Third-Party Complaint
2. Memorandum in Support (if separate event required)
3. Proposed Third-Party Complaint (attachment or separate event per court rule)
4. Defendant Declaration in Support
5. Exhibit Index (Exhibits A–M)
6. Exhibit Packet (single PDF bookmarked A-M if permitted)
7. Proposed Order
8. Certificate of Service

## File Naming Convention

Use consistent sortable names:

1. 01_Motion_Joinder_Third_Party_Quantum.pdf
2. 02_Memo_Support_Joinder_Quantum.pdf
3. 03_Proposed_Third_Party_Complaint_Quantum.pdf
4. 04_Declaration_Barber_Cortez.pdf
5. 05_Exhibit_Index_Quantum.pdf
6. 06_Exhibits_A_to_M_Quantum.pdf
7. 07_Proposed_Order_Joinder_Quantum.pdf
8. 08_Certificate_of_Service_Quantum.pdf

## If Court Requires Combined Filing

If one merged filing is required, combine in this order:

1. Motion
2. Memorandum
3. Declaration
4. Exhibit Index
5. Exhibits
6. Proposed Third-Party Complaint
7. Proposed Order
8. Certificate of Service

## Optional Bench Copy Order

For hearing binders or bench copy:

1. Cover sheet with one-paragraph relief request.
2. One-page hearing outline.
3. Motion and memorandum.
4. Key exhibits first (C, D, E, G, I), then remaining exhibits.
5. Proposed order on top of final tab.

## Upload Validation Checklist

1. Caption and case number match across all files.
2. Exhibit labels in declaration match exhibit tabs exactly.
3. Service certificate includes all required recipients and method.
4. PDF text is legible, rotated correctly, and not password-locked.
