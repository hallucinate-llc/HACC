## Technical Specifications
### Language
- Python 3.8+
- Modular design for easy extension
### Dependencies
- `requests` (HTTP downloads)
- Standard library (json, csv, pathlib, logging, etc.)
- System tools: `pdftotext` (poppler), optional `ocrmypdf` + `tesseract`
### Data Flow
```
Brave/CommonCrawl/Google Search
↓
download_manager.py (dedupe & download)
↓
research_results/documents/raw/*.pdf
↓
parse_pdfs.py (extract text)
↓
research_results/documents/parsed/*.txt + *.json
↓
index_and_tag.py (keyword match & score)
↓
research_results/document_index.json + summary.csv
↓
report_generator.py (summarize)
↓
FINDINGS_*.txt + DETAILED_REPORT_*.txt
```
### Scoring Rubric
| Score | Criteria | Action |
|-------|----------|--------|
| 0 | No DEI/proxy language | Compliant |
| 1 | DEI/proxy only | Review context |
| 2 | DEI/proxy + binding indicators | Prioritize review |
| 3 | Explicit DEI policy + mandatory | Immediate remediation |
### Keywords Indexed
**DEI**: diversity, equity, inclusion, dei, deia, deib, underrepresented, underserved
**Proxies**: cultural competence, lived experience, diversity statement, safe space, bipoc, minority-only, first-generation, low-income targeting, equity plan
**Binding**: policy, ordinance, statewide, contract, required, must, shall, mandatory, applicable to

## How to Use (Three Paths)
### Path A: Manual Evidence Collection
No API keys needed. Collect documents manually, then analyze.
```bash
cp documents/*.pdf research_results/documents/raw/
python3 research_data/scripts/run_collection.py --pdf-dir research_results/documents/raw/
```
### Path B: Automated Web Search
Use Brave Search API to discover Oregon.gov documents.
```bash
export BRAVE_API_KEY="your_key"
python3 research_data/scripts/run_collection.py
```
### Path C: Hybrid
Combine manual PDFs + API search for comprehensive coverage.

## Support & Customization
### To Add New Keywords
Edit `index_and_tag.py`, update the keyword lists in `DocumentIndexer` class.
### To Change Risk Scoring
Edit `_calculate_risk_score()` method in `index_and_tag.py`.
### To Modify Report Format
Edit `report_generator.py` to customize output templates.
### To Add New Search API
Create new collector script (e.g., `collect_cc.py` for Common Crawl), import in `run_collection.py`.
